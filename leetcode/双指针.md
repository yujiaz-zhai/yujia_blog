# 双指针

#### 1、核心思想

双指针的朴素算法：

```c++
for(int i=0; i<n; i++){
  for(int j=0; j<n; j++){
    ...
  }
}
```

朴素算法的复杂度是$O(n^2)$的，双指针的优化算法是将复杂度降到$O(n)$，**如下（模版）**：

```c++
for(int i=0, j=0; i<n; i++){
  while(j<i && check(i, j)){
    j++
  }
  ...
}
```

 还可以考虑这种形式：

```
int l=0, r=len;
while(l<r){
	...
}
```
